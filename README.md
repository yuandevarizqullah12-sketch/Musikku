# 🎵 Musikku — YouTube Music Player

A modern, minimal, and fast web music player that uses YouTube as its source. Built with vanilla JavaScript, Vercel serverless functions, and Firestore for caching.

## Features
- Search songs via YouTube Data API
- Caching with Firestore and browser localStorage
- Sticky bottom player with play/pause, seek, volume, repeat (off/one/all)
- Media Session API integration (lock screen controls)
- Responsive dark theme with skeleton loading
- Modular ES6 code structure
